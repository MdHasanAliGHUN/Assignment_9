# Assignment 09 - WarmPaws: Winter Pet Care

## Overview
WarmPaws একটি cozy winter companion platform যা পোষা প্রাণীর জন্য তৈরি। এই ওয়েবসাইটে ব্যবহারকারী পেতে পারেন: Local winter pet care services, Winter clothing & grooming options, Expert tips for pet health. SPA হিসেবে তৈরি, React + Firebase + Tailwind CSS + React Hook Form ব্যবহার করে।  

## Live LINk
https://warmpaws-petcare-c80f8.web.app/
## Features
- Authentication: Email/Password & Google Sign-In  
- Forgot Password: Redirect to Gmail  
- Profile Management: Update name & photo  
- Responsive Design: Mobile, Tablet, Desktop  
- Animations: AOS, Swiper.js  
- SPA Behavior: No errors on route reloads  
- Services: Display JSON-based winter pet services  
- Booking: Book service form with toast notifications  

## Tech Stack
React, React Router, Firebase Authentication, Tailwind CSS, React Hook Form, AOS, Swiper.js, React-Spring,  React Icons
